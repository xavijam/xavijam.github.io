Javier Álvarez Medina

UI developer at Vizzuality

In 2009, while I was finishing my Computer Scientist Degree at Rey Juan Carlos university in Madrid, I started as intern in Vizzuality.

At the beginning I learnt how to develop small RIAs with Flash technology, but after some time and projects done, I realized my real passion was pointing to the Front-End development.

Anti-flash self-confessed, I love new web technologies and I’m always ready to try them out. Focus on HTML, CSS (SASS, LESS, Compass,…), JavaScript (Backbone, D3.js, Node, …), maps libraries (Google Maps, Leaflet, OpenLayers,…) and hardly ever Ruby.

Today I work as UI Developer transforming wireframes into real web applications, and I’m putting all my efforts and love on CartoDB, a geospatial database in the cloud.

I am interested neither in a new job nor in big and paid conferences. I only trust meetups where people are worried to help other people, selflessly.

Behind the scenes

Some open frameworks, plugins or libraries I’ve used working on this page

  • Jekyll

    Engine for building static websites from dynamic components.
  • Compass

    Open-source CSS Authoring Framework.
  • Backbone.js

    JS framework for structuring JS as MVC.
  • jQuery

    JS library for manipulating HTML.
  • Pure

    Set of small, responsive CSS modules.
  • D3.js

    JS library for handling docs based on data.
  • Modernizr

    JS library that detects HTML5&CSS3 features.
  • Spin.js

    Creates a JS spinning activity indicator.
  • Ladda

    UI concept using buttons with loaders.
